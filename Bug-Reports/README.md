# Bug Reports

This folder contains sample bug reports created during manual testing.

## Contents

- Login Bugs
- Registration Bugs
- Product Search Bugs
- Shopping Cart Bugs
- Checkout Bugs
- Order Management Bugs

## Bug Report Format

Each bug report includes:

- Bug ID
- Module
- Summary
- Environment
- Steps to Reproduce
- Expected Result
- Actual Result
- Severity
- Priority
- Status
- Screenshot (if applicable)
